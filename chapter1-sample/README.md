## Vue Cliのインストール
npm install -g @vue/cli

## Vue cli-service-globalのインストール
npm install -g @vue/cli-service-global

## プロトタイプの実行
適用な場所にvueファイルを作成し、その階層までコマンドプロンプトで移動して以下のコマンドを実行
vue serve

## Vue Cliでプロジェクト作成
vue create プロジェクト名

## プロジェクトの実行
npm run serve

## プロジェクトをビルド(dist配下に出力)
npm run build

## Vue CliをGUIで実行
vue ui
